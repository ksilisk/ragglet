Чанкинг (Chunking) — это процесс разбиения больших документов на мелкие фрагменты для индексации. Recursive и Semantic — две основные стратегии с разными подходами к сохранению контекста.

Ключевые термины: Chunk (фрагмент), Recursive Chunking (рекурсивное разбиение), Semantic Chunking (семантическое разбиение), Splitter (разделитель).

Главные тезисы:

Recursive Chunking использует иерархию разделителей (\n\n, \n, точка, пробел), пытаясь сохранить цельность предложений и абзацев.
Semantic Chunking использует эмбеддинги для определения границ между фрагментами: там, где резко меняется смысл (тема), создается новый чанк.
Выбор стратегии зависит от структуры документа: для статей с четкими абзацами подойдет Recursive, для диалогов или сложных текстов — Semantic.
