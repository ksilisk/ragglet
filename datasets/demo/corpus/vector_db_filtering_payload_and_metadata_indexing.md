Payload Filtering (Фильтрация по метаданным) — это возможность выполнять поиск векторов с дополнительными условиями по полям-атрибутам (автор, дата, категория), хранящимся вместе с вектором.

Ключевые термины: Payload (метаданные), Metadata Indexing (индексирование метаданных), Pre-filtering (предварительная фильтрация), Post-filtering (пост-фильтрация).

Главные тезисы:

Pre-filtering: сначала применяются фильтры по метаданным, потом поиск по векторам в отфильтрованном подмножестве. Медленно при селективных фильтрах.
Post-filtering: сначала ищутся ближайшие векторы, потом отсеиваются неподходящие по метаданным. Быстро, но можно потерять релевантные результаты.
Для эффективности нужно индексировать часто используемые поля метаданных (аналогично индексам в обычных БД).
