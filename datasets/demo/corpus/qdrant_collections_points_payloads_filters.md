Qdrant — векторная БД с открытым исходным кодом. Collections, Points, Payloads — ее базовые понятия: коллекция содержит точки (векторы с ID), к которым можно прикрепить payload (метаданные) и фильтровать по ним.

Ключевые термины: Collection (коллекция), Point (точка), Payload (нагрузка/метаданные), Filter (фильтр), Qdrant.

Главные тезисы:

Collection — аналог таблицы, определяет размерность векторов и метрику расстояния.
Point — единица хранения: уникальный ID, вектор и опциональный payload (JSON-объект с метаданными).
Фильтры в Qdrant поддерживают сложные условия (AND/OR/NOT) по полям payload и позволяют комбинировать их с векторным поиском (pre-filtering).
