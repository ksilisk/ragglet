Self-Query и Metadata Filtering — это методы интеллектуального поиска, при которых система автоматически извлекает из запроса пользователя не только текстовый поисковый запрос, но и структурные фильтры для атрибутов (метаданных).

Ключевые термины: Metadata (метаданные), Filtering (фильтрация), Structured Query (структурированный запрос), Self-Querying Retriever.

Главные тезисы:

Позволяет задавать вопросы типа: «Найди документы по теме X, опубликованные после 2023 года».
LLM парсит запрос и разделяет его на две части: семантический поиск («X») и фильтр по метаданным («дата > 2023»).
Это резко повышает точность поиска, сужая пространство для поиска только релевантным набором документов по заданным критериям.
