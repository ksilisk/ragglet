Document Structure Parsing — это извлечение текста из сложных форматов (PDF, HTML) с сохранением их логической структуры (заголовки, таблицы, списки) для последующего чанкинга.

Ключевые термины: Parser (парсер), Layout Analysis (анализ макета), Markdown Conversion (конвертация в маркдаун), Document Hierarchy (иерархия документа).

Главные тезисы:

Простое извлечение текста из PDF часто ломает порядок абзацев и теряет заголовки, делая чанки бессмысленными.
Современные парсеры (Unstructured, LlamaParse) восстанавливают структуру: определяют заголовки, объединяют ячейки таблиц, сохраняют вложенность списков.
Правильный парсинг критически важен: чанки, созданные с учетом иерархии документа, дают гораздо более релевантный поиск.
