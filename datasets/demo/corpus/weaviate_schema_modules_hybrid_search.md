Weaviate — векторная БД со встроенными модулями и гибкой схемой данных. Ключевая особенность — нативный гибридный поиск (векторный + BM25) с автоматической нормализацией весов.

Ключевые термины: Class (класс), Property (свойство), Module (модуль), Hybrid Search (гибридный поиск), Weaviate.

Главные тезисы:

Schema определяет классы объектов и их свойства (с типами данных), аналогично коллекциям.
Modules расширяют функциональность: векторизаторы (модули эмбеддингов), модули генерации, модули для ранжирования.
Hybrid Search комбинирует векторный поиск и BM25 с настраиваемым параметром alpha (0 — только BM25, 1 — только векторы).
