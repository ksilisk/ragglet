Vectorizer Modules в Weaviate — это встроенные интеграции с моделями эмбеддингов (как локальными, так и API), которые автоматически векторизуют данные при импорте и запросы при поиске.

Ключевые термины: Vectorizer (векторизатор), Module (модуль), Inference API (API для инференса), *text2vec-* (префикс модулей), *multi2vec-*.

Главные тезисы:

Модули text2vec-openai, text2vec-cohere, text2vec-transformers автоматически превращают текст в векторы.
При импорте Weaviate вызывает указанный модуль для генерации эмбеддингов и сохраняет их вместе с объектом.
При поиске запрос автоматически векторизуется тем же модулем, обеспечивая семантическое соответствие.
