Batching (пакетная обработка), задержка (latency) и пропускная способность (throughput) — ключевые показатели производительности при работе с моделями эмбеддингов в реальных приложениях.

Ключевые термины: Batch Size (размер пакета), Latency (задержка), Throughput (пропускная способность), Inference (инференс), GPU Memory (память GPU).

Главные тезисы:

Обработка текстов батчами (пакетами) увеличивает throughput, но увеличивает latency для первого элемента в пакете.
Оптимальный batch size зависит от модели и доступной памяти GPU: слишком большой пакет может вызвать OOM (out of memory).
Для real-time RAG критична низкая latency, поэтому часто выбирают компромисс: небольшой batch size или выделенные инстансы.
