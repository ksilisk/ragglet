Query Expansion (Расширение запроса) — это техника улучшения поиска путем генерации нескольких вариаций исходного пользовательского запроса для более полного охвата релевантной информации.

Ключевые термины: Multi-Query, Paraphrasing (перефразирование), Query Transformation, Diversity (разнообразие).

Главные тезисы:

LLM используется для создания 3-5 альтернативных формулировок одного и того же вопроса пользователя.
Все сгенерированные запросы выполняются параллельно, результаты собираются и дедуплицируются.
Это повышает полноту (recall) поиска, особенно когда исходный запрос короткий или неоднозначный, снижая риск пропустить важный документ.
