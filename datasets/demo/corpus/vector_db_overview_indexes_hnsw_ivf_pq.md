Vector Database — это специализированная база данных для эффективного хранения и поиска векторных эмбеддингов. Индексы (HNSW, IVF, PQ) — это структуры данных, ускоряющие приблизительный поиск ближайших соседей (ANN).

Ключевые термины: ANN (Approximate Nearest Neighbor), HNSW (Hierarchical Navigable Small World), IVF (Inverted File Index), PQ (Product Quantization).

Главные тезисы:

Точный линейный поиск (kNN) невозможен при больших объемах данных — нужны приблизительные методы (ANN) с жертвой точности ради скорости.
HNSW — графовый метод, обеспечивающий высокую скорость и точность, но требующий много памяти.
IVF + PQ — комбинация кластеризации (IVF) и сжатия (PQ), позволяющая экономить память ценой небольшой потери качества.
