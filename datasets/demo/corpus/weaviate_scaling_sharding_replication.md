Weaviate поддерживает горизонтальное масштабирование через шардирование (sharding) и репликацию (replication) в кластерной конфигурации для обеспечения отказоустойчивости и производительности.

Ключевые термины: Sharding (шардирование), Replication Factor (коэффициент репликации), Cluster (кластер), Node (узел), Raft Consensus.

Главные тезисы:

Данные распределяются по шардам на основе хеша ID объекта (consistent hashing).
Репликация создает копии шардов на разных узлах; количество копий задается replicationFactor.
Кластер использует Raft для координации и выбора лидера, обеспечивая консистентность метаданных.
