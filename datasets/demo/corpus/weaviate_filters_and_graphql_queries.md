Weaviate использует GraphQL в качестве основного языка запросов, что позволяет комбинировать векторный поиск, фильтры по метаданным и агрегации в одном элегантном запросе.

Ключевые термины: GraphQL, Get Query (запрос получения), Aggregate Query (запрос агрегации), Where Filter (фильтр), BM25 Search.

Главные тезисы:

Get запросы с nearText/nearVector выполняют векторный поиск с опциональными фильтрами where.
BM25 запросы используют ключевой поиск по текстовым полям.
Фильтры поддерживают операторы сравнения (Equal, LessThan, Like) и логические операторы (And, Or), работая с индексированными свойствами.
