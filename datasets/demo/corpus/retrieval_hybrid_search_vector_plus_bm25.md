Гибридный поиск (Hybrid Search) — это комбинация семантического (векторного) и ключевого (лексического) подходов для повышения качества поиска в RAG.

Ключевые термины: Vector Search (поиск по эмбеддингам), BM25 (алгоритм лексического поиска), Ensemble (ансамбль), Sparse/Dense vectors.

Главные тезисы:

Векторный поиск хорош для учета смысла, но может пропустить точное совпадение терминов.
BM25 отлично работает с ключевыми словами и идентификаторами, но не понимает синонимы.
Гибридный поиск объединяет результаты обоих методов, компенсируя недостатки каждого и обеспечивая более надежное извлечение контекста.
