Context Assembly — это этап формирования финального промпта для LLM: собранные чанки оборачиваются в шаблон с инструкциями и ссылками на источники (цитатами).

Ключевые термины: Prompt Template (шаблон промпта), Citation (цитирование), Source Attribution (атрибуция источника), Assembly Strategy (стратегия сборки).

Главные тезисы:

Шаблон должен четко отделять контекст от запроса и содержать инструкцию: «отвечай только на основе текста ниже».
Добавление метаданных (название документа, страница) к каждому чанку позволяет LLM давать ответы с цитатами для проверяемости.
Стратегия сборки включает сортировку чанков по релевантности и фильтрацию низкокачественных фрагментов перед подачей в модель.
