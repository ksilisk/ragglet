Pgvector — расширение для PostgreSQL, добавляющее поддержку векторов и векторного поиска. Поддерживает два типа индексов: IVFFlat (приблизительный поиск) и HNSW (точный графовый поиск).

Ключевые термины: Extension (расширение), IVFFlat (инвертированный файл), HNSW, Lists (списки), Probes (зонды), Pgvector.

Главные тезисы:

IVFFlat — классический кластерный индекс. Параметры: lists (количество кластеров, обычно sqrt(N)) и probes (количество просматриваемых кластеров при поиске).
HNSW — графовый индекс. Параметры: m (количество связей) и ef_construction (при построении).
Индексы создаются как обычные PostgreSQL индексы: CREATE INDEX ON items USING ivfflat (embedding vector_cosine_ops)
