Настройка pgvector требует понимания как параметров самого расширения, так и общесистемных настроек PostgreSQL (work_mem, maintenance_work_mem, autovacuum) для обеспечения производительности.

Ключевые термины: work_mem (память для операций сортировки), maintenance_work_mem, autovacuum (автоочистка), effective_cache_size, shared_buffers.

Главные тезисы:

work_mem важен для построения HNSW-индексов — недостаточный размер приводит к использованию диска (temp files) и замедлению.
Регулярный ANALYZE необходим для обновления статистики, влияющей на планировщик запросов.
Индексы IVFFlat требуют периодической перестройки (REINDEX) при значительных изменениях данных для поддержания точности.
