RAG Failure Modes — это типовые сценарии, при которых система Retrieval-Augmented Generation дает сбой. Их принято делить на проблемы поиска (Retrieval) и проблемы генерации (Generation).

Ключевые термины: Low Precision (низкая точность), Low Recall (низкая полнота), Hallucination (галлюцинация), Ignoring Context (игнорирование контекста).

Главные тезисы:

Ошибки поиска: ретривер вернул нерелевантные чанки (низкая точность) или пропустил важные (низкая полнота).
Ошибки генерации: LLM проигнорировала поданный контекст и выдала ответ из своих «знаний» или дополнила его выдуманными фактами.
Для диагностики failures нужно оценивать поиск и генерацию по отдельности.
