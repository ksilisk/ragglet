Milvus — мощная векторная БД с богатой структурой данных. Collections (коллекции), Partitions (партиции) и Indexes (индексы) — три уровня организации данных для эффективного управления и поиска.

Ключевые термины: Collection (коллекция), Partition (партиция), Index (индекс), Segment (сегмент), Milvus.

Главные тезисы:

Collection — логическая группа векторов с общей размерностью и метрикой (аналог таблицы).
Partitions — подразделы внутри коллекции, позволяющие сужать поиск (например, по дате) и ускорять фильтрацию.
Индексы строятся на сегментах внутри партиций и критически важны для производительности ANN-поиска.
